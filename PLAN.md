# IT대구 강의시간표 웹 서비스 — 목차

> **기술 스택:** Flask · Pandas · Vanilla JS · Bootstrap 5  
> **작성일:** 2026-04-08  
> **목표:** CSV 기반 강의 시간표를 웹에서 깔끔하게 조회하고 원하는 과정을 빠르게 검색하는 뷰어

## 기능 목록

| # | 분류 | 기능명 | 한줄 설명 |
|---|------|--------|-----------|
| F1 | 핵심 | 시간표 그리드 뷰 | 시간대 × 강의실 격자 형태로 시각화 |
| F2 | 핵심 | 과정 목록 뷰 | 파싱된 전체 과정을 카드 리스트로 조회 |
| F3 | 핵심 | 과정 검색 | 과정명 / 강사 / 강의실 키워드 실시간 필터 |
| F4 | 핵심 | 과정 상세 팝업 | 클릭 시 해당 과정의 모든 정보를 모달로 표시 |
| F5 | 핵심 | 월 / 평일·주말 전환 | 10개 파일(4~8월 × 평일/주말) 간 탭 전환 |
| F6 | 핵심 | 오늘 날짜 하이라이트 | 현재 진행 중·오늘 개강 과정 시각적 강조 |
| N1 | 부가 | 강사·강의실 필터 | 드롭다운으로 특정 강사·강의실만 표시 |
| N2 | 부가 | 열 기준 정렬 | 목록 뷰에서 컬럼 클릭 시 오름/내림차순 정렬 |
| N3 | 부가 | 인쇄 최적화 | 불필요한 UI 숨기고 시간표만 깔끔하게 인쇄 |
| N4 | 부가 | 다크 모드 | 화면 밝기 토글, 설정 저장 |
