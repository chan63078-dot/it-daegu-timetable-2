## F1. 시간표 그리드 뷰

**목적:** CSV 원본 구조를 그대로 웹 테이블로 재현하되, 가독성을 높인다.

**화면 구성**

```
        │ A[10층]-1      │ A[10층]-2      │ B[10층]-1      │ ...
────────┼────────────────┼────────────────┼────────────────┼────
09:00   │ 빅데이터 분석   │                │ AI 활용 소프트  │
        │ 플랫폼 개발자   │                │ 웨어 개발자     │
        │ 석득화 | 월~금  │                │ 최해만 | 월~금  │
        │ 04/21 ~ 05/20  │                │ 04/09 ~ 05/20  │
        │ 배정 5/20명 ●  │                │ 배정 14/20명 ● │
10:00   │      ↑ 계속    │ 나만의 GPT     │      ↑ 계속    │
        │   (rowspan)    │ 올인원 로드맵  │   (rowspan)    │
...
19:00   │ AI엔지니어      │ ...            │                │
        │ LLM멀티모달1   │                │                │
```

**세부 규칙**
- 셀 병합 (`rowspan`): 동일 과정이 여러 시간 블록에 걸치면 병합하여 표시
- 빈 셀: 연한 회색 배경 (텍스트 없음)
- 강의실 그룹별 헤더 배경색 구분 (A동/B동/C동/AI아카데미/이스트캠프 등)
- 셀 안 표시 내용 (우선순위 순):
  1. 과정명 (굵게, 넘치면 말줄임)
  2. 강사명 | 요일
  3. 개강일 ~ 종강일
  4. 배정 N/정원N명 + 충족률 색상 배지
- 가로 스크롤: 강의실이 많을 때 상단 헤더 행 고정(sticky)
- 세로 스크롤: 좌측 시간 열 고정(sticky)

**인터랙션**
- 셀 클릭 → F4 과정 상세 모달 열림
- 마우스 호버 → 과정명 툴팁 (말줄임 시 전체 표시)

## F2. 과정 목록 뷰

**목적:** 검색·필터 결과를 카드 형태로 훑어보기 쉽게 나열한다.

**화면 구성**

```
┌──────────────────────────────────┐  ┌──────────────────────────────────┐
│ [진행중] 빅데이터 분석 플랫폼 개발  │  │ [예정] AI 활용 소프트웨어 개발자   │
│ 강의실  A[10층]-1                 │  │ 강의실  B[10층]-1                 │
│ 강사    석득화                    │  │ 강사    최해만3                   │
│ 기간    2026-04-21 ~ 2026-05-20  │  │ 기간    2026-04-09 ~ 2026-05-20  │
│ 시간    09:00 ~ 13:30  월~금      │  │ 시간    09:00 ~ 13:30  월~금      │
│ 배정    5 / 20명  ████░░ 25%     │  │ 배정    14 / 33명  ████████ 42%  │
└──────────────────────────────────┘  └──────────────────────────────────┘
```

**세부 규칙**
- 기본 정렬: 개강일 오름차순
- 상태 배지 4종:
  - `진행중` (초록): 오늘 날짜가 개강일~종강일 사이
  - `오늘개강` (파랑): 오늘이 개강일
  - `예정` (회색): 개강일이 오늘 이후
  - `종료` (연한 회색): 종강일이 오늘 이전
- 배정 현황 프로그레스 바:
  - 80% 이상 → 빨강 (마감 임박)
  - 50~79% → 주황
  - 50% 미만 → 초록
- 검색 결과 수 표시: "총 32개 중 5개 표시"
- 검색어 매칭 부분 **하이라이팅**

**인터랙션**
- 카드 클릭 → F4 과정 상세 모달
- 빈 결과: "검색 결과가 없습니다" 안내 메시지

## F3. 과정 검색

**목적:** 과정명·강사·강의실 키워드로 원하는 과정을 빠르게 찾는다.

**화면 구성**

```
🔍 [                                   과정명 / 강사 / 강의실 검색  ✕ ]
   총 32개 중 5개 표시
```

**세부 규칙**
- 검색 대상: 과정명, 강사명, 강의실명 (동시 검색, OR 조건)
- 실시간 필터: 입력 후 300ms 디바운스로 즉시 반영
- 대소문자·공백 무시
- 검색어 2자 미만이면 전체 목록 표시
- `✕` 버튼으로 검색어 초기화
- 시간표 뷰 상태에서 검색 시 → 자동으로 목록 뷰로 전환
- URL 쿼리 파라미터에 검색어 반영 (`?q=파이썬`) → 링크 공유 가능

**검색 범위**
- 현재 선택된 월/구분(평일·주말) 내에서만 검색
- 전체 월에 걸친 검색은 N1 필터 확장으로 지원

## F4. 과정 상세 팝업 (모달)

**목적:** 특정 과정의 모든 정보를 한 화면에서 깔끔하게 확인한다.

**화면 구성**

```
┌─────────────────────────────────────────────────────┐
│  [진행중]  빅데이터 분석 플랫폼 개발자 양성과정 1    ✕ │
├─────────────────────────────────────────────────────┤
│  강의실     A[10층]-1                               │
│  강사       석득화                                  │
│  요일       월~금                                   │
│  수업시간   09:00 ~ 13:30                           │
│  개강일     2026-04-21                              │
│  종강일     2026-05-20                              │
├─────────────────────────────────────────────────────┤
│  정원       26명 (수강인원 20명)                     │
│  배정       5명                                     │
│  전체 출석율  0%   ░░░░░░░░░░                       │
├─────────────────────────────────────────────────────┤
│  비고       휴강: 5/1, 5/5                          │
└─────────────────────────────────────────────────────┘
```

**세부 규칙**
- 모달 열기: 그리드 뷰 셀 클릭 또는 목록 뷰 카드 클릭
- 모달 닫기: `✕` 버튼 / `ESC` 키 / 모달 바깥 영역 클릭
- 빈 필드(강사 미입력 등)는 행 자체를 숨김
- 비고 필드: 줄바꿈 그대로 표시 (휴강일, 보강일 등)
- 모바일: 하단 시트(bottom sheet) 형태로 전환

## F5. 월 / 평일·주말 전환

**목적:** 10개 파일을 탭으로 빠르게 이동한다.

**화면 구성**

```
[4월] [5월] [6월] [7월] [8월]          ← 월 탭 (오늘 해당 월 자동 선택)
      [평일]  [주말]                    ← 구분 탭
```

**세부 규칙**
- 초기 로드 시 오늘 날짜 기준으로 해당 월 자동 선택
  - 해당 월이 범위 밖(4~8월 외)이면 가장 가까운 월 선택
- 파일이 존재하지 않는 월·구분 조합은 탭 비활성화(disabled) 처리
- 탭 전환 시:
  1. 로딩 스피너 표시
  2. API 재호출 (`/api/timetable?month=4&type=weekday`)
  3. 그리드·목록 뷰 동시 갱신
  4. 검색어는 유지
- URL 파라미터에 반영 (`?month=4&type=weekday`) → 새로고침 후에도 상태 복원

## F6. 오늘 날짜 하이라이트

**목적:** 오늘 기준으로 현재 상황을 즉시 파악할 수 있게 한다.

**상황별 처리**

| 상황 | 시간표 뷰 | 목록 뷰 |
|------|----------|---------|
| 오늘 개강 | 셀 상단에 `🟢 오늘 개강` 배지 | 카드에 `오늘개강` 파란 배지 |
| 진행 중 | 셀 배경 옅은 초록 + 테두리 강조 | 카드에 `진행중` 초록 배지 |
| 종료 | 셀 배경 연한 회색 + 과정명 취소선 | 카드에 `종료` 회색 배지 + 흐리게 |
| 예정 | 기본 스타일 | 카드에 `예정` 회색 배지 |

- 헤더 우측에 오늘 날짜 항상 표시: `오늘: 2026-04-08 (수)`
- 진행 중 과정이 있을 경우 상단 요약 배너 표시:
  ```
  📌 현재 진행 중인 과정 12개 | 오늘 개강 2개
  ```

## N1. 강사·강의실 필터 (부가)

**목적:** 특정 강사 또는 강의실의 시간표만 빠르게 조회한다.

**화면 구성**

```
강사 [전체 ▾]   강의실 [전체 ▾]   [필터 초기화]
```

**세부 규칙**
- 강사·강의실 드롭다운은 현재 탭 데이터 기준으로 동적 생성
- 검색(F3)과 AND 조건으로 결합 (검색어 + 강사 + 강의실 모두 만족)
- 시간표 뷰: 선택된 강의실 열만 표시, 나머지 열 숨김
- 필터 초기화 버튼으로 전체 복원

## N2. 열 기준 정렬 (부가)

**목적:** 목록 뷰에서 원하는 기준으로 과정을 정렬한다.

**정렬 기준**

| 컬럼 | 기본 방향 |
|------|----------|
| 과정명 | 가나다 오름차순 |
| 개강일 | 오름차순 (기본) |
| 종강일 | 오름차순 |
| 배정인원 | 내림차순 |
| 출석율 | 내림차순 |

- 컬럼 헤더 클릭 → 오름차순, 재클릭 → 내림차순, 재재클릭 → 정렬 해제
- 현재 정렬 기준 아이콘으로 표시 (▲ / ▼)

## N3. 인쇄 최적화 (부가)

**목적:** 시간표를 종이로 출력할 때 깔끔하게 나오게 한다.

**규칙**
- `인쇄` 버튼 클릭 또는 `Ctrl+P` 시 인쇄 레이아웃 적용
- 인쇄 시 숨김: 검색바, 탭, 버튼류, 배너
- 인쇄 시 표시: 시간표 그리드 + 현재 선택된 월/구분 제목
- 가로 방향(landscape) 권장 안내 문구 출력
- CSS `@media print` 로 별도 스타일 정의

---

## N4. 다크 모드 (부가)

**목적:** 야간 사용 환경에서 눈의 피로를 줄인다.

**규칙**
- 헤더 우측 `🌙 / ☀️` 토글 버튼
- Bootstrap 5 다크 테마 (`data-bs-theme="dark"`) 활용
- 선택값은 `localStorage`에 저장 → 새로고침 후에도 유지
- 시스템 설정(`prefers-color-scheme`) 자동 감지로 초기값 결정
